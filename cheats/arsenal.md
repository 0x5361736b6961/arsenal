# Arsenal

% arsenal, internal

## List global variable
```
>show
```

## Clear all global variable
```
>clear
```

## Set commons global variable
```
>set LHOST=<LHOST> LPORT=<LPORT> RHOST=<RHOST> RPORT=<RPORT>
```

## Set ip global variable
```
>set ip=<IP>
```

## Set custom global variable
```
>set <GLOBALVAR>=<value>
```

## Exit
```
>exit
```
